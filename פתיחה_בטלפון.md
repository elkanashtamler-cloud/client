# איך לפתוח את האפליקציה בטלפון

המחשב והטלפון **חייבים** להיות באותה רשת Wi‑Fi (אותו Router).

---

## שלב 1: הפעלת השרת עם גישה מרשת

בטרמינל (במחשב), מתוך תיקיית `client`:

```bash
npm run dev:mobile
```

(במקום `npm run dev` — הפקודה `dev:mobile` מאפשרת גישה מהטלפון.)

אחרי ההפעלה יופיעו שתי כתובות, למשל:
- **Local:** http://localhost:5173/
- **Network:** http://192.168.1.105:5173/

---

## שלב 2: מציאת הכתובת (Network)

בטרמינל חפש את השורה שמתחילה ב־**Network:**.  
הכתובת תיראה בערך כך: `http://192.168.1.XXX:5173/`  
(המספרים XXX משתנים לפי הרשת שלך.)

---

## שלב 3: פתיחה בטלפון

1. וודא שהטלפון מחובר **לאותה רשת Wi‑Fi** כמו המחשב.
2. פתח בדפדפן בטלפון (Chrome, Safari וכו').
3. בשורת הכתובת הקלד בדיוק את מה שמופיע ב־**Network**, למשל:
   ```
   http://192.168.1.105:5173
   ```
4. היכנס לאתר — תופיע מסך ההתחברות של רשימת הקניות. התחבר עם אותו אימייל וסיסמה כמו במחשב.

---

## אם האתר לא נפתח בטלפון

### 1. בדיקות בסיסיות
- **אותו Wi‑Fi:** הטלפון והמחשב חייבים להיות באותה רשת (אותו Router).
- **השרת רץ:** וודא שבטרמינל רצה `npm run dev:mobile` ואתה רואה "Local" ו-"Network".
- **כתובת נכונה:** בטלפון הקלד **בדיוק** את הכתובת מ-**Network** (כולל `http://` ומספר ה־port `:5173`).

### 2. חומת האש של Windows (הסיבה הנפוצה)

Windows חוסם לעיתים חיבורים מהרשת ל־port 5173. צריך לאפשר.

**אופציה א — הוספת כלל (מומלץ):**
1. במחשב: לחץ **Windows** והקלד **Windows Defender Firewall**.
2. בחר **"Advanced settings"** / **"הגדרות מתקדמות"**.
3. בצד שמאל: **Inbound Rules** → **New Rule...**.
4. בחר **Port** → Next.
5. **TCP**, **Specific local ports:** הקלד `5173` → Next.
6. בחר **Allow the connection** → Next.
7. סמן את שלוש התיבות (Domain, Private, Public) → Next.
8. שם: למשל `Vite Dev 5173` → Finish.

**אופציה ב — בדיקה זמנית:**  
השבת זמנית את חומת האש ל-**Private network** (הגדרות Windows → Privacy & Security → Windows Security → Firewall) ונסה שוב מהטלפון. אם עובד — הבעיה היא החומת אש; החזר את החומת אש והשתמש באופציה א.

### 3. אם עדיין לא עובד
- **נסה דפדפן אחר** בטלפון.
- **רענן** את הדף או מחק את הכתובת והקלד שוב.
- **וודא שאין VPN** על המחשב או על הטלפון שמפריד ביניהם.
- ב־Router: אם יש **"Client isolation"** או **"AP isolation"** — כבה (אם יש לך גישה להגדרות).

---

**טיפ:** אחרי שנכנסת פעם אחת מהטלפון, אפשר להוסיף את הדף ל־**מסך הבית** (Chrome: תפריט → "הוסף למסך הבית") כדי לפתוח כמו אפליקציה.
