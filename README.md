# 🛒 Smart Family Shopping List

A real-time, Progressive Web App (PWA) designed to make family grocery shopping seamless and perfectly synchronized.

## ✨ Key Features
* **Real-Time Synchronization:** Items added, checked off, or deleted are instantly updated across all connected devices.
* **Supermarket Mode:** A unique live status feature. When one person is at the store, they can toggle "Supermarket Mode" to notify family members to add any last-minute items.
* **PWA & Mobile Ready:** Fully installable on iOS and Android home screens for a native, full-screen app experience.
* **Cloud Database:** Secure and fast data management.

## 🛠️ Tech Stack
* **Frontend:** React, Vite, TypeScript, Tailwind CSS
* **Backend & Database:** Supabase (PostgreSQL, Realtime Subscriptions)
* **Deployment:** Vercel
