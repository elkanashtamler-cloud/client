# מה לעשות עכשיו — צעד אחר צעד

## שלב א': קובץ .env (חשוב)

האפליקציה קוראת מפתחות מקובץ בשם **`.env`** (טקסט פשוט), לא מקובץ Word.

1. בתיקייה **`client`** צור קובץ חדש בשם: **`.env`**  
   (ב-VSCode/Cursor: קובץ חדש → שמור בשם → `.env`)
2. פתח את **`client.env.docx`** (או את הדשבורד של Supabase) והעתק:
   - את ה-**Project URL**
   - ואת מפתח ה-**anon public**
3. בתוך קובץ **`.env`** הדבק בדיוק את שני השורות האלה (עם הערכים שלך):

```
VITE_SUPABASE_URL=https://xxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6...
```

שמור את הקובץ.

## שלב ב': הרצת הפרויקט

1. פתח **טרמינל** (בתוך Cursor: Terminal → New Terminal, או PowerShell).
2. עבור לתיקיית הפרויקט והכנס לתוך `client`:

```bash
cd "c:\Users\elkan\OneDrive\Desktop\פרוייקטים אישיים\פרוייקט קניות\client"
```

3. התקן את החבילות (פעם אחת):

```bash
npm install
```

4. הפעל את שרת הפיתוח:

```bash
npm run dev
```

5. בדפדפן פתח את הכתובת שמופיעה (בדרך כלל **http://localhost:5173**).

---

## שלב ג': משתמש להתחברות

אם עדיין לא יצרת משתמש ב-Supabase:

- היכנס ל-Supabase → **Authentication** → **Users** → **Add user** → **Create new user**
- הזן אימייל וסיסמה (אותם תשתמשו בשני המכשירים).

בדפדפן התחבר עם אותו אימייל וסיסמה.

---

**סיכום:** קובץ `.env` בתוך `client` → `npm install` → `npm run dev` → פתח localhost:5173 → התחבר.
