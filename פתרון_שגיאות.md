# פתרון שגיאות בטרמינל

## השגיאה: "npm is not recognized" / "npm לא מזוהה"

**משמעות:** המחשב לא מוצא את Node.js ואת npm. צריך להתקין Node.js.

### התקנת Node.js (Windows)

1. היכנס לאתר: **https://nodejs.org**
2. הורד את הגרסה **LTS** (כפתור ירוק).
3. הרץ את קובץ ההתקנה והמשך Next עד הסוף.
4. **חשוב:** סגור את Cursor (או את הטרמינל) ופתח מחדש — רק אז הטרמינל יזהה את `node` ו-`npm`.
5. פתח טרמינל חדש ובדוק:
   ```bash
   node -v
   npm -v
   ```
   אם מופיעים מספרי גרסה — ההתקנה הצליחה.

6. אחר כך הרץ שוב:
   ```bash
   cd "c:\Users\elkan\OneDrive\Desktop\פרוייקטים אישיים\פרוייקט קניות\client"
   npm install
   npm run dev
   ```

## שגיאות אחרות

אם אתה מקבל **הודעת שגיאה אחרת** (למשל אדומה בטרמינל):

1. **העתק את כל טקסט השגיאה** (או צלם מסך).
2. **הדבק כאן בצ'אט** — אז אוכל לומר בדיוק מה לתקן.

שגיאות נפוצות:
- **"VITE_SUPABASE_URL is undefined"** — לא נוצר קובץ `.env` או שהשמות בו לא נכונים (חייבים `VITE_SUPABASE_URL` ו-`VITE_SUPABASE_ANON_KEY`).
- **"Cannot find module"** — אחרי שינוי בקוד או עדכון חבילות, נסה שוב: `npm install` ואז `npm run dev`.
- **שגיאות רשת / CORS** — וודא ש-Project URL ומפתח anon נכונים ב-`.env` ושהטבלאות ב-Supabase קיימות.

---

**סיכום:** אם השגיאה היא ש-npm לא מזוהה — התקן Node.js מ-nodejs.org, סגור ופתח מחדש את Cursor, ואז הרץ `npm install` ו-`npm run dev`. אם השגיאה שונה — העתק והדבק אותה כאן.
